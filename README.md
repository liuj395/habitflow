# HabitFlow - 习惯流

一款专业的习惯戒除与培养应用，帮助用户建立积极的生活方式。

## 🌟 主要功能

### 📝 习惯管理
- **添加习惯**: 支持培养类和戒除类习惯
- **分类筛选**: 按类型查看不同习惯
- **智能提醒**: 帮助用户坚持习惯

### 📊 进度追踪
- **每日打卡**: 简单直观的完成标记
- **连续天数**: 实时统计坚持天数
- **可视化图表**: 7天进度趋势图
- **完成统计**: 总完成次数记录

### 🏆 激励系统
- **成就徽章**: 6种不同成就等待解锁
- **积分奖励**: 完成习惯获得积分
- **里程碑**: 重要节点自动庆祝
- **实时通知**: 成就解锁即时提醒

### 📈 数据分析
- **今日统计**: 当日完成情况概览
- **完成率**: 习惯执行效率分析
- **趋势分析**: 个人进步轨迹
- **详细报告**: 单个习惯深度分析

## 📱 使用指南

### 快速开始
1. **添加习惯**: 在顶部输入框中输入习惯名称
2. **选择类型**: 选择"培养习惯"或"戒除习惯"
3. **每日打卡**: 点击右侧圆形按钮完成打卡
4. **查看进度**: 点击习惯卡片查看详细统计

### 习惯类型说明
- **培养习惯**: 需要坚持做的积极行为（如运动、阅读）
- **戒除习惯**: 需要避免的消极行为（如熬夜、吸烟）

### 成就系统
- **初学者**: 添加第一个习惯
- **坚持一周**: 连续完成7天
- **月度达人**: 连续完成30天
- **多面手**: 同时管理5个习惯
- **完美一周**: 一周内完成所有习惯
- **百分达人**: 获得100积分

## 🔧 安装部署

### 本地运行
```bash
# 克隆项目
git clone [项目地址]

# 进入目录
cd habitflow

# 启动本地服务器
python -m http.server 8000
# 或使用其他静态服务器

# 访问应用
open http://localhost:8000
```

### PWA 安装
1. 在支持的浏览器中打开应用
2. 点击地址栏的"安装"按钮
3. 确认安装到设备桌面
4. 享受原生APP般的体验

## 💡 使用建议

### 习惯培养技巧
1. **从小习惯开始**: 选择容易坚持的小目标
2. **保持一致性**: 每天同一时间执行
3. **环境设计**: 创造有利的执行环境
4. **及时奖励**: 完成后给自己小奖励

### 戒除习惯策略
1. **识别触发因素**: 了解不良习惯的触发点
2. **替代行为**: 用积极行为替代消极行为
3. **环境改变**: 移除诱发因素
4. **寻求支持**: 告诉朋友你的目标
